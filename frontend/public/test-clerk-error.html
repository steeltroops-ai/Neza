Test Clerk Error Handling

This page helps you test the Clerk error handling implementation. Click the button below to simulate a Clerk "Invalid host" error.

The error will be displayed as:

{
  "errors": [
    {
      "message": "Invalid host",
      "long_message": "We were unable to attribute this request to an instance running on Clerk. Make sure that your Clerk Publishable Key is correct.",
      "code": "host_invalid"
    }
  ]
}

Our implementation should catch this error and display a user-friendly message instead of the raw JSON.
